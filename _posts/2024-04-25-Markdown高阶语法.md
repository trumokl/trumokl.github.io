> “简单是最终的复杂。 ”



下面是一些高级 Markdown 语法特性：



### 1. 常用的html标签

+ **删除线**

  代码：

  ```
  <del>此处为删除线</del>  或 ~~此处为删除线~~
  ```

  效果 ：

  <del>此处为删除线</del>

+ **上下标**

  代码：

  ```
  <sup>此处为上标</sup> 或 ^此处为上标^
  <sub>此处为下标</sub> 或 ~此处为下标~
  ```

  效果：

  <sup>此处为上标</sup>            <sub>此处为下标</sub>      

+ **链接**

  代码：

  ```
  <a herf="https://www.google.com/">此处为google地址</a>
  ```

  效果：

  <a herf="https://www.google.com/">此处为google地址</a>
